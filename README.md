# RSSL XML Trace Viewer Tools

The utility was designed for decoding and displaying the value from RSSL XML Tracing log generated by RFA C++/.NET and EMA/ETA C/C++. The main purpose of the project is to create WPF GUI application to display data for MRN Real-Time News provided by RIC MRN_STORY. Anyway, the core functionality of the application can be used to decode the value of the field entry data inside the XML element. Therefore, the application can be used to open RSSL tracing log which contains data from other message model type such as Market Price, Market By Price and Market By Order. This utility should be able to help verify data issue or confirm the Market Price data or MRN Real-Time news data that API sends and receive from Provider Server or TREP component. Please note that RSSL Tracing log and this utility was provided for troubleshooting purpose only.

For the full article, please refer to the [RSSL XML Trace Viewer Tools](https://developers.lseg.com/en/article-catalog/article/rssl-xml-trace-viewer-tools) article on the [LSEG Developer Community](https://developers.lseg.com/en).
